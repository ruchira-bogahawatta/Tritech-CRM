How to run the TRI-TECH CRM  Project


Download the  zip file (Porject)


Extract the file and copy tritech-crm folder


Paste inside root directory(For xampp xampp/htdocs)


Start Apache and MySQL servers from XAMPP Control Panel.


Open PHPMyAdmin (http://localhost/phpmyadmin)


Create a database with name tritech-crm


Import tritech-crm.sql file (Given inside the SQL file folder in tritech-crm folder)


Run the script http://localhost/tritech-crm/ (Will be redirect to the login page)
